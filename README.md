# OlaRide-Dashboard
# 🚖 Ola Ride Analytics Dashboard

An interactive data analytics dashboard built using **Streamlit + Python + Plotly** to analyze ride patterns, revenue, cancellations, and customer behavior.

🔗 **Live App:** 
https://olaride-dashboard-mlyyltn37oezyswuvcj7cq.streamlit.app/
---

## 📊 Key Features

* 📈 Ride trends analysis (daily, hourly)
* 📊 Booking status breakdown
* 🚗 Vehicle performance insights
* ❌ Cancellation reason analysis
* 💰 Revenue by payment method
* 👤 Customer and driver rating insights

---

## 🧠 Business Insights

This dashboard helps answer key questions like:

* What are the peak ride hours?
* Which vehicle types generate the most revenue?
* Why are rides getting canceled?
* How does customer satisfaction vary?

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

---

## 📂 Dataset

* Simulated / cleaned Ola ride dataset
* Includes ride datetime, booking status, vehicle type, revenue, ratings, etc.

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run App.py
```
